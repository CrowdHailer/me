Web without the framework

Live coding session to build a chat appliction using Elixir. This demonstration makes use of Elixir core utilities and Raxx. It aims to show how capababile Elixir is for web development without the additional complexity of a framework Raxx being a Webinterface layer designed to promote simplicity and functional purity.

I gave this talk for the Elixir London meetup group.
