Ace an Education in TCP

Ace is a TCP server that was built with the design priority of having welcoming source code. I wanted to learn more about the network layer and how to work with erlang/OTP.

This talk shows the path from simple echo functionality to a robust general purpose server. The challenges and encountered along the way. And why Elxir was such a perfect fit for this project.

It may also be possible to present this or similar as an more technical introduction to why BEAM to other communities.
